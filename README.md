# rosae
CLI for rosa

## Command

```sh
rosae generate username # Generate a key pair in ~/.rosa/

rosae friends # Give you your friends list

rosae friend friend_name # Give you the public of your friend

rosae public # give your public key

rosae add friend_name encoded_key # Add a friend to your friends list

rosae delete friend_name # Delete a friend from your friends list

rosae help # duh.
```
